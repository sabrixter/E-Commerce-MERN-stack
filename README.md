# eCommerce Backend

An Express.js application for managing eCommerce operations.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

For development with auto-reload:

```bash
npm run dev
```
